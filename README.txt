Description:
   	generic DnD character maker
	will ask if you want to create a textfile for your character once done

Requirements:
	g++ compiler
	requires c++20 or higher
	
Building:
	use the "make" command in the directory to build the executable
	use "make clean" to then remove the temporary files

Usage:
	launch the executable with "./dndsheet"
	if you decide to create the textfile, it will be created in the executable directory
